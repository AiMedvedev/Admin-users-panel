# Admin-users-panel
Админ-панель для работы с данными пользователей через JSON-server.

## Запуск
В консоли ввести 


## Используются: 
JavaScript, Webpack, JSON-server.

## Реализовано: 
- модульный подход,
- добавление/редактирование/удаление пользователей,
- поиск по имени в БД,
- фильтры по правам доступа и наличию детей.
